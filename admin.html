Flujo de trabajo
  1. Añade o edita proyectos en la sección Proyectos
  2. Los cambios se guardan automáticamente en tu navegador (localStorage)
  3. Cuando termines, haz clic en Exportar JSON
  4. Reemplaza el archivo data/projects.json con el descargado
  5. El sitio web mostrará los cambios inmediatamente en este navegador

Se usa como thumbnail en la galería y en el hero de inicio

URLs externas: usa links de Unsplash, CDN, etc.
Archivos locales: coloca los archivos en assets/images/projects/ y usa la ruta relativa.
Drag & drop: arrastra una imagen sobre el campo URL para convertirla a Base64 (solo para imágenes <800KB).
Imágenes 360°: necesitas una imagen equirectangular (relación de aspecto 2:1, típicamente 4000×2000px o superior).
Servicios como Matterport, Ricoh Theta o cámaras 360° producen este formato directamente.
YouTube: usa la URL de embed (https://www.youtube.com/embed/VIDEO_ID).
Local: coloca el archivo .mp4 en assets/video/ y usa la ruta relativa.
